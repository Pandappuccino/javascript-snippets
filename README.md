# Pandappuccino's JavaScript Snippets
This repository contains small snippets of JavaScript that I've put together for personal use that can be reused for multiple websites by linking to them with <script src=""></script> tags!

## Simple Tab Toggles
A lightweight tab layout script that lets users create web pages with tabbed sections that can be switched between with a simple click, using custom data attributes and classes! Check out [the sample page](https://codepen.io/editor/Pandappuccino/pen/019d8751-905a-7841-9e06-89ea3212cc43) to see them in use!
